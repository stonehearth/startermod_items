Welcome to Stonehearth! Here are the patch notes for this build.

Error window

The error window is much improved and can now be closed permanently.

Start up diagnostics

We've added a lot more diagnostics to help determine the cause of crashes on start up. Although this build may still crash on start up, the data it provides will help us fix the issue.

Bugs fixed

  • Lua consuming 100% cpu usage after mutiple errors.
  • Removing a building after starting construction generates scaffolding error.
  • Deleting a building after placing template throws errors.
  • Items acquired from trader failing to restock.
  • Unable to promote when talisman is located at high altitude.
  • Items in stockpiles at high altitude not included in inventory or score.
  • Hitting escape while planting banner throws UI error.
  • Errors when shepherd is chasing down sheep.
  • Waterfall error when water bodies are merging.

Other changes

  • Party menu now has hotkey 'g'.
  • Various tooltips fixed
